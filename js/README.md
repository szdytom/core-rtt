# JavaScript/TypeScript Code for Core RTT

This directory contains the JavaScript/TypeScript code. This is a monorepo root for all JavaScript/TypeScript code related to the project.

- `js/core-autotest/`: the autotesting framework built for testing core game logic (C++) and corelib functionalities.
- `js/corereplay/`: a library for deserializing binary replay files generated by the game engine.
- `js/replay-cli`: a command-line tool for analyzing replay files, built on top of `corereplay`.

## Building and testing

The JavaScript packages in this directory are managed with pnpm. To install dependencies, build and test the packages, you can run the following commands:

```sh
cd js
pnpm install
pnpm run build
pnpm run typecheck
pnpm run test
```
